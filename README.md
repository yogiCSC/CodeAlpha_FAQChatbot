# 🤖 FAQ Chatbot

This project was developed as part of the CodeAlpha Artificial Intelligence Internship.

The FAQ Chatbot is an AI-based application developed using Python and Streamlit. It answers user questions using predefined FAQs and NLP techniques like cosine similarity.

---

## 🚀 Features

- Interactive chatbot interface
- Answers frequently asked questions
- Simple and user-friendly design
- Fast response generation

---

## 🛠 Technologies Used

- Python
- Streamlit
- scikit-learn
- nltk

## ▶️ How to Run the Project

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python -m streamlit run chatbot.py
```
